# Privacy Policy / Polityka Prywatności
_Last updated: 2025-12-02_

## PL Polityka Prywatności

### 1. Informacje ogólne
Ta aplikacja jest projektem open-source udostępnianym na licencji GPL. Aplikacja **nie zbiera, nie przechowuje ani nie przetwarza żadnych danych osobowych użytkowników na własnych serwerach**.

### 2. Dane lokalizacyjne
Aplikacja może prosić o dostęp do lokalizacji urządzenia (np. `ACCESS_COARSE_LOCATION`, `ACCESS_FINE_LOCATION`) wyłącznie w celu zapewnienia prawidłowego działania jej funkcji. Wszystkie dane lokalizacyjne są przetwarzane **wyłącznie na urządzeniu użytkownika** i **nie są przesyłane** do twórców aplikacji ani na żaden serwer aplikacji.

### 3. Google Maps SDK
Aplikacja korzysta z komponentu **Google Maps SDK**, który może automatycznie przetwarzać dane takie jak:
- adres IP,
- dane o urządzeniu,
- przybliżona lub dokładna lokalizacja (jeśli użytkownik wyrazi zgodę).

Dane te są przetwarzane **bezpośrednio przez Google LLC** zgodnie z ich Polityką Prywatności: https://policies.google.com/privacy

### 4. Brak udostępniania danych
Aplikacja **nie udostępnia żadnych danych** twórcom aplikacji ani osobom trzecim.

### 5. Open-source
Kod źródłowy aplikacji jest publicznie dostępny w repozytorium GitHub, umożliwiając użytkownikowi zweryfikowanie sposobu działania aplikacji.

### 6. Kontakt
W razie pytań można skontaktować się poprzez stronę repozytorium GitHub.

## EN Privacy Policy

### 1. General Information
This application is an open-source project distributed under the GPL license. The application **does not collect, store, or process any personal data on its own servers**.

### 2. Location Data
The app may request access to device location (e.g., `ACCESS_COARSE_LOCATION`, `ACCESS_FINE_LOCATION`) solely to enable its functionality. All location data is processed **only on the user’s device** and is **not transmitted** to the developers or any servers.

### 3. Google Maps SDK
The application uses **Google Maps SDK**, which may automatically process data such as:
- IP address,
- device information,
- approximate or precise location (with permission).

This data is processed **directly by Google LLC** under their Privacy Policy: https://policies.google.com/privacy

### 4. No Data Sharing
The application **does not share any personal or location data** with developers or third parties.

### 5. Open-source
The app’s source code is publicly available in the GitHub repository, allowing users to review how the application works.

### 6. Contact
For questions regarding this privacy policy, please contact us through the GitHub repository page.
